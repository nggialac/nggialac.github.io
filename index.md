## Những ngôn ngữ và công nghệ đã va choạm
-C/C++: Nhập môn lập trình cơ bản + Cấu trúc dữ liệu và giải thuật; Ban đầu đối với mấy môn này thì còn lập trình kiểu hướng cấu trúc.

-Về Java: hầu hết chương trình học đều là Java vì nó là ngôn ngữ mã nguồn mở (2007) và được rất nhiều lập trình viên sử dụng. Mặt khác, nó thuần hướng đối tượng, có thể dễ dàng build các ứng dụng Web, Mobile,... Với chương trình đào tạo của trường, các môn lập trình ứng dụng đều code Java (Spring Framework cho Website, API còn Android cho mobile, thậm chí còn dùng Java Swing để code WinApp và môn Kỹ thuật đồ họa).

-C#: có tiếp xúc thông qua một số môn bắt buộc (1 phần do thầy), làm quen với DevExpress, Binding Data trên GridView, Report, ASP.NET MVC...

-SQL Server: quản lý Database, phân tán CSDL (Distributed) - sấp mặt với đồ án.

-HTML/CSS, JS tự mò :D
